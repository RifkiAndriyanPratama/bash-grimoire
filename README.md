## Apa itu Shell
Sebuah program yang menyediakan sebuah antarmuka berbasis teks untuk berinteraksi dengan sistem operasi

- bash (bourne again shell)
- zsh, fish, ksh (kornshell) --> sebuah alternatif
- powershell 

## Apa itu Shell Scripting
Proses menulis perintah dalam sebuah script, dan ketika dijalankan akan menjalankan semua perintah yang ada didalam script tersebut yang mana memungkinkan user untuk:

- otomasi
- proses (pengolahan data)
- pipeline (membuat alur kerja)

## Mengapa Shell Scripting Ini Penting
Karena dalam efisiensi tugas yang biasanya diselesaikan berjam-jam dapat diselesaikan dalam hitungan menit bahkan detik, mengurangi human error karena semua dikelola dalam script.

## Bagaimana Cara Kerjanya
Shell scripting ini bekerja dengan cara membaca dan menjalankan perintah secara berurutan
konsep dasar:

- perintah dasar (ls, cd, grep)
- variable
- logika dan decision making
- looping
- rederection (mengalirkan sebuah output ke perintah lain (| >))
